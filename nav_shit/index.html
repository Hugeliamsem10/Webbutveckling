Header Main Nav Article Aside Div Section Footer

<header>

A header is a section at the top of a webpage or document that introduces the content. In HTML, it usually contains titles, logos, or navigation links and helps organize and identify the page or section.

<main>

In HTML, main is a semantic element that represents the primary content of a webpage. It contains the main information that is directly related to the pages topic, excluding things like headers, footers, and navigation menus.

<nav>

In HTML, nav is a semantic element used to contain navigation links. It usually includes menus or links that help users move to different pages or sections of a website.

<article>

In HTML, article is a semantic element used to represent independent, self-contained content, such as a blog post, news article, or forum post. It is meant for content that can stand on its own.

<aside>

In HTML, aside is a semantic element used for content that is related to the main content but not part of it. It is often used for sidebars, extra information, ads, or related links.

<div>

In HTML, div is a generic container element used to group and organize other elements. It has no specific meaning on its own and is often used for layout, styling, or structuring content with CSS and JavaScript.

<section>

In HTML, section is a semantic element used to group related content together. It usually represents a distinct part of a webpage, often with its own heading.

<footer>

In HTML, footer is a semantic element used to define the bottom section of a webpage or a section. It typically contains information like copyright, contact details, or links.
